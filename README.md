# Password checker CLI

A command-line password checker built in Go.

## Features
- Supprt special character case values in password string.
- Support number case values in password string.
- Support uppercase and lowercase values in password string. 

## Usage
- make sure the go is installed 
``` go version```
``` 
go version go1.26.4
```
- git clone the repo
- Run
```go run main.go```
